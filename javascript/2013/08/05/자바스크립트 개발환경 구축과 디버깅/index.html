자바스크립트 개발환경 구축과 디버깅

준비물

  • 텍스트 에디터 이클립스, 에디트 플러스, 메모장 등 자신에게 가장 편한 텍스트 에디터를 사용하면 됩니다.
  • 파이어폭스, 파이어버그 플러그인 이 문서의 내용을 따라가기 위해서는 반드시 파이어폭스와 파이어버그 플러그인을 설치해야 합니다.

파이어폭스, 파이어버그 설치하기

  • 모질라 파이어폭스 사이트에서 다운로드 받습니다. http://www.mozilla.or.kr/ko/firefox/
  • 다운로드 받은 파일을 실행해서 설치합니다.
  • 파이어폭스 프로그램을 실행합니다.
  • 메뉴에서 [도구] - [부가기능]을 선택합니다.
  • [전체검색] 탭을 선택하고, Firebug 를 검색해서 설치합니다.
  • 파이어폭스를 닫고 다시 시작합니다.

Hello World!

  • 이제 텍스트 에디터를 이용해서 helloworld.html 파일을 생성하고, 아래 코드를 입력하고 저장합니다.
<html>
<head>
<title>Hello World</title>
</head>
<body>
<script type="text/javascript">
console.log("Hello World!");
</script>
  • helloworld.html 파일을 파이어폭스에서 불러옵니다.(탐색기에서 파이어폭스로 드래그)
  • 파이어폭스 우측 하단에 벌레 아이콘을 클릭해서 파이어버그를 실행합니다. 그리고 콘솔 탭의 화살표를 눌러 Enable 상태로 바꿉니다.
  • 새로고침을 하게 되면 콘솔창에 Hello World!라고 출력됩니다.

지금까지 파이어버그의 콘솔창을 이용해서 Hello World를 출력해봤습니다. 여기서 중요한 것은 console.log를 이용한 디버깅입니다. 위의 예제는 String 값을 출력하는 것이지만 Object나 Array를 넣게 되면 안에 들어가는 내용이 무엇인지 보기 좋게 나오게 됩니다. 이러한 디버깅 방법은 Opera의 기본 디버거인 잠자리나 크롬의 개발자도구에서 별도의 플러그인 없이 지원하며, IE8 개발자도구에서도 String까지는 지원합니다. 게다가 모바일 사파리에서도 디버그 설정을 활성화 시키면 console.log를 사용해서 디버깅할 수 있습니다.
