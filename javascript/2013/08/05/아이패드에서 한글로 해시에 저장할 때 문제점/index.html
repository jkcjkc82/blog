아이패드에서 한글로 해시에 저장할 때 문제점

location.hash = "name=한글"; 이런식으로 입력을 하게 되면 아이패드 사파리에서는 자동으로 인코딩이 되는데,

location.href = location.href+"#name=한글"

이런식으로 입력하면 한글이 인코딩 안되게 저장이 된다.
